ACHUSP
==================

Sistema de achados e perdidos para a Universidade de São Paulo

Instalação
==========

Requisitos:

ImageMagick  
GSL (Gnu Scientific Library)  
Ruby 1.9.2  
JAVA 6 ou superior  

Instalando:

```bash
bundle install
```
Rodando:

```bash
rails server
```

Iniciando o solr em ambiente de desenvolvimento
```bash
rake sunspot:solr:start
